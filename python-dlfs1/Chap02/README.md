# Chap02: 퍼셉트론

### 퍼셉트론
**퍼셉트론**은 다수의 신호를 입력으로 받아 하나의 신호를 출력한다.  
```math
y = \begin{cases}
0 & (w_1x_1 + w_2x_2 \leq \theta) \\
1 & (w_1x_1 + w_2x_2 > \theta)
\end{cases}
```
각각의 입력 신호에 고유한 가중치를 부여해, 결과에 주는 영향력을 조절한다. 앞으로 위 식에서 $\theta$ 대신 $-b$로 치환해 좌변으로 정리하여 표현하면 $wx + b > 0$ 꼴이 될 것이다.  

### 퍼셉트론의 한계
단순 퍼셉트론으로는 XOR을 나타낼 수 없다는 한계가 있었지만, **다층 퍼셉트론**을 사용하여 문제를 해결했다.
